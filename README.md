# Описание проекта

LMS (Learning Management System) - это платформа для онлайн-обучения с возможностью
создания курсов, уроков, управления подписками и приема платежей.

## Основные возможности

1. Управление пользователями: регистрация, JWT авторизация, профили
2. Курсы и уроки: создание, редактирование, удаление с разграничением прав
3. Платежи: интеграция со Stripe, подписки на курсы
4. Уведомления: email рассылки при обновлении курсов
5. Фоновые задачи: Celery + Redis для асинхронной обработки
6. Документация API: Swagger UI и ReDoc
7. Docker: полная контейнеризация проекта

## Установка и запуск

- Python 3.11+
- PostgreSQL 15+
- Redis 7+
- Docker (опционально)

## Локальная установка

Клонирование репозитория
```
it clone https://github.com/v1rtefl-coder/django-rf.git
cd django-rf
```

## Создание виртуального окружения
```
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

## Установка зависимостей
```
pip install -r requirements.txt
```
